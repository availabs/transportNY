// methodology
How these numbers are produced

Annual rollups are computed from NPMRDS probe data submitted to FHWA per 23 CFR § 490. Per-year values match FHWA-published targets within ±0.05 pp; small discrepancies stem from data-version cutoffs (NPMRDS v1 → v2 in 2017, v2 → v2.1 in 2022) which are noted in each table's y/y column.
