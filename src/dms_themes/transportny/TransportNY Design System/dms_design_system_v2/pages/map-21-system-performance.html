// 03 How MAP-21 targets work

Reading "met" the way FHWA does.

Significant progress = a two-prong test

FHWA counts a measure as making significant progress if either condition holds — not target alone:

A

Meets or beats the target — actual is at least as good as the 2-yr / 4-yr target.

B

Better than baseline — actual improved over the period's baseline year, even if short of target (TTTR: by ≥ 0.01).

Met target Significant progress Not meeting
Performance periods

Four-year periods with a 2-yr midpoint check and a 4-yr final check. Current period P2 · 2022–2025; next P3 · 2026–2029.

MPO adoption · 180 days

Within 180 days of the State setting a target, each MPO either supports the State target or sets its own. NY MPOs support the statewide targets.

Urban congestion is UZA-only

PHED and Non-SOV apply only to large UZAs in nonattainment/maintenance — in NY, New York-Newark and Poughkeepsie-Newburgh.
