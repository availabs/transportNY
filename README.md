## Transport NY

https://transportny.org

AVAILabs client for TransportNY platform.

##### Installation

Create a `src/config.json` file using `src/config.json.template`.

```sh
$ git clone git@github.com:availabs/transportNY.git
$ cd transportNY
$ git submodule init
$ git submodule update
$ npm install
$ npm start
```
